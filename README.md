# 💫 About Me:

🔧 Embedded Software Engineer passionate about building end-to-end systems — from hardware-level design to scalable backend services and real-time applications.

⚙️ I enjoy working across the stack:
- Developing firmware for IoT devices using platforms like ESP32, STM32, and Raspberry Pi
- Designing and prototyping hardware with PCB design and 3D CAD tools
- Building desktop applications with JavaFX, Full Stack applications using Spring Boot, Laravel, MERN stack, and microservices architecture
- Containerizing and deploying applications using Docker

🌐 I have hands-on experience with connected systems and communication technologies:
- Wireless & connectivity: WiFi, LoRa, GSM, GPRS
- IoT protocols: MQTT, Zigbee, and other lightweight communication protocols
- Real-time communication: WebSockets, Socket.IO, and event-driven architectures

🌱 Currently exploring:
- Advanced microservices and distributed system design
- Scalable IoT architectures and cloud integration
- Real-time data processing and device communication at scale

🚀 What defines my work:
- I enjoy bridging hardware and software, making complete systems work seamlessly
- I focus on practical implementations, debugging real-world problems
- I enjoy exploring how different systems communicate and scale together

🤝 Looking for:
- Opportunities to work on production-grade IoT and distributed systems
- Guidance and collaboration in system design and scalable architectures

💬 Ask me about:
Embedded Systems • IoT • Spring Boot • MERN Stack • Docker • MQTT • Real-Time Systems • PCB Design

⚡ Fun fact:
I enjoy building systems where hardware, backend, and real-time communication come together — from sensor to server to user.
